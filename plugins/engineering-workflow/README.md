# Engineering Workflow Plugin

This plugin provides workflow agents for prompt distillation, technical design analysis, and implementation.

## Contents

### Agents

- **Distiller**: Distills a high-level prompt into a clear, actionable problem statement with acceptance criteria.
- **Technical Analyst**: Analyzes the problem statement to identify key components, dependencies, and potential challenges.
- **Implementer**: Generates a structured implementation plan, breaking down the problem into manageable tasks and suggesting appropriate tools, libraries, or frameworks.

### Skills

- **Realign**: Identifies and reports inconsistencies in code patterns across the codebase, helping to maintain a coherent engineering workflow.
